## DataOptics – Multi-Tenant SaaS Platform

**DataOptics** is a SaaS platform designed specifically for **optical store management**.

The platform supports multiple businesses operating on a **single application instance** while maintaining **complete data isolation per client**.

### Multi-Tenant Architecture

- Single Symfony application
- Single backend
- Dedicated database per tenant
- Full data isolation between clients

This architecture allows the platform to scale while maintaining security and data separation.

### Core Features

- patient management
- optical prescriptions
- sales and invoicing
- inventory management
- reporting and analytics

### Tech Stack
